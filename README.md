## 项目简介 / Project Overview

这是一个在桌面右键菜单中添加和删除快捷启动项的小工具。

This is a small tool to add and remove quick launch items from the desktop context menu.

### 主要功能 / Key Features
- ✅ 添加程序到右键菜单 / Add programs to context menu
- ✅ 从右键菜单移除程序 / Remove programs from context menu
- ✅ 简单易用的界面 / Simple and user-friendly interface
