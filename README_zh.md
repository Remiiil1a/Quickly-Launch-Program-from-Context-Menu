## 使用方法

### 添加程序到右键菜单
1. 以管理员身份运行程序
2. 点击"添加程序"按钮
3. 选择要添加到右键菜单的可执行文件
4. 程序将出现在桌面右键菜单中

### 从右键菜单移除程序
1. 以管理员身份运行程序
2. 在列表中选择要移除的程序
3. 点击"移除"按钮
